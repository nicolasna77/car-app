## Environment

# Database

```
DB_HOST=localhost
DB_PORT=Port
DB_USER=User
DB_PASSWORD=Password
DB_NAME=db_Name
```

# API

```
NEXT_PUBLIC_API_URL=http://localhost:3000
NEXT_PUBLIC_API_PORT=3000
```

## Start

# API

```bash
npm run dev:nodemon
```

# Frontend

```bash
npm run dev
```
